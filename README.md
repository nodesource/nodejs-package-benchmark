# nodejs-package-benchmark

This package allows you to benchmark different runtimes using popular
packages operations.

## Supported packages

- [x] fastify
- [x] lodash
- [x] prettier
- [ ] webpack
- [x] babel
- [x] moment
- [x] dotenv
- [x] pinojs
- [x] winston
- [ ] helmet
- [ ] node-sass
- [x] underscore
- [ ] minify
