# nodejs-package-benchmark

This package allows you to benchmark different runtimes using popular
packages operations.

## Supported packages

- [x] fastify
- [x] lodash
- [ ] prettier
- [ ] webpack
- [ ] babel
- [ ] moment
- [ ] dotenv
- [ ] pinojs
- [ ] winston
- [ ] helmet
- [ ] node-sass
- [ ] underscore
- [ ] minify
